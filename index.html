PROJECTS

ALPR

In my final year at university, I built an Automated License Plate Recognition System (ALPR) as a team project. Existing ALPR systems had low feasibility in the Indian context, where standard violations of license plate recommendations are common. We attempted to target this issue with a multi-stage application. I learned a lot about Computer Vision from this project and gained insights into major hurdles for such systems, like privacy laws hindering data acquisition. We had a tough time given the time constraint for development and optimization, but we managed to achieve decent results.

Color Identifier

The older version of my website had background images. I wanted to blend my color theme with the image boundaries, and for this I needed to retrieve the hexcode of what I was looking at. Rather than use online tools, I decided to turn this into a fun project. I used the mouse event callbacks feature available in the OpenCV library for Python. The application uses a list of color names and returns the closest match for a clicked point on an image. I am currently working on adding more features like argument parsers for command line use and keyboard event handlers.
